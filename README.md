# Networking Basics for Cyber Security

### Objective
To understand basic networking concepts and analyze live network traffic using Wireshark.

### Tools Used
- Wireshark

### Concepts Covered
- IP Address
- MAC Address
- DNS
- TCP and UDP
- TCP Three-Way Handshake
- HTTP vs HTTPS

### Task Performed
- Captured live network traffic using Wireshark
- Applied protocol filters (HTTP, DNS, TCP)
- Observed TCP three-way handshake
- Identified plain-text and encrypted traffic
- Captured and analyzed DNS queries

### Files Included
- analysis_report.txt – Detailed observations and analysis
- task3_network_capture.pcapng – Packet capture file

### Outcome
Gained practical knowledge of packet sniffing and network traffic analysis.
